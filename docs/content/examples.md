# Examples

## 基本解析

```bash
cargo run -- -d ~/Downloads
```

## 隠しファイル込み

```bash
cargo run -- -d ~/Downloads -a
```

## examples ディレクトリ

LeSort には以下のサンプルが含まれています。

* basic_analysis.rs
* detailed_analysis.rs
* score_calculation.rs
* hidden_files.rs

実行例:

```bash
cargo run --example basic_analysis
```
