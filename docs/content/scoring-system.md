# Scoring System

LeSort はディレクトリの整理状態を数値化します。

評価に使用する情報:

* ファイル総数
* ファイル種類数
* 種類の分散度

## スコア範囲

| Score  | Evaluation |
| ------ | ---------- |
| 80-100 | Excellent  |
| 60-79  | Good       |
| 40-59  | Fair       |
| 0-39   | Poor       |

高得点ほど整理されたディレクトリであることを示します。
